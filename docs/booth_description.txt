【タイトル】
SteamVR Tracker Jitter Analysis Tool - トラッカー揺れ解析・可視化ツール

【概要】
VIVE Trackerやコントローラーの位置・回転ジッタ（微細な揺れ）をリアルタイムで可視化・解析するツールです。
ベースステーション(BS)の設置環境の確認や、トラッキング精度の検証、不具合の切り分け（どのBSが原因か等）に最適です。

本ツールはPython製ですが、実行ファイル(.exe)形式で配布しているため、環境構築不要ですぐにご利用いただけます。

【主な機能】
■ リアルタイムグラフ表示
・位置 (X, Y, Z) と 回転 (Pitch, Yaw, Roll) を同時プロット
・グラフはウィンドウサイズに合わせて自動調整
・不要なデバイスの非表示が可能

■ 統計解析
・直近500サンプルの標準偏差(σ)をリアルタイム計算し表示
・トラッキングロス率(%)の監視

■ ベースステーション監視 (Drift Monitor)
・ベースステーション自体の位置ズレ（振動や熱膨張によるドリフト）を検知し警告
・各トラッカーからBSまでの距離を表示

■ データ保存
・計測データをCSV形式で保存可能（MATLAB等での解析用）

【内容物】
・SteamVR_Jitter_Tool.exe (実行ファイル)

【使い方】
1. SteamVRを起動し、トラッカーの電源を入れます。
2. ツール(exe)を起動します。
3. 「Start Measurement」ボタンを押して計測を開始します。
4. 左側のリストから表示したいデバイスにチェックを入れます。
5. 必要に応じて「Save CSV」でデータを保存します。
